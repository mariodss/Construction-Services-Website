# Construction-Services-Website

## Project Overview
This project involved building a responsive and user-friendly business website for a real client. The main goal was to improve the companyâ€™s online presence and provide visitors with clear information about services, contact options, and an intuitive user experience.

## Tech Stack
- WordPress
- Bluehost Hosting
- WordPress Themes & Plugins
- Page Builder (Elementor)

## My Role
I was responsible for the full website development and design process. This included setting up WordPress on Bluehost, configuring the theme, customizing layouts, implementing frontend styling, and ensuring responsiveness across devices.

## Key Features
- Custom WordPress theme configuration and styling
- Fully responsive design (mobile, tablet, desktop)
- Optimized navigation and page structure
- Contact form integration
- SEO-friendly structure
- Performance and usability improvements

## Hosting & Deployment
- Hosting provider: Bluehost  
- WordPress installation and configuration  
- Domain and basic hosting setup  
- Site deployment and testing
